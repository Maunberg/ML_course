PCA

Принцип работы коротко

Проекция на направления максимальной дисперсии.

Математическое обоснование подробно

Собственные векторы ковариационной матрицы.

Собственная реализация
C=np.cov(Xc,rowvar=False); vals,vecs=np.linalg.eigh(C)
Реализация sklearn
PCA(n_components=10).fit_transform(X)
Описание гиперпараметров

n_components, whiten.

Интуиция использования

Шумоподавление и быстрая визуализация.

Сетка гиперпараметров для гридсёрч
{"pca__n_components":[2,5,10,20],"pca__whiten":[False,True]}

Truncated SVD

Принцип работы коротко

SVD без центрирования, подходит sparse данным.

Математическое обоснование подробно

X≈U_kΣ_kV_k^T.

Собственная реализация
U,S,VT=np.linalg.svd(X,full_matrices=False)
Реализация sklearn
TruncatedSVD(n_components=100).fit_transform(X)
Описание гиперпараметров

n_components, n_iter.

Интуиция использования

Стандарт для TF-IDF и больших разреженных матриц.

Сетка гиперпараметров для гридсёрч
{"svd__n_components":[50,100,200],"svd__n_iter":[5,10]}

FastICA

Принцип работы коротко

Разделяет независимые источники сигнала.

Математическое обоснование подробно

Максимизация негауссовости компонент.

Собственная реализация
# whiten + fixed-point iterations for unmixing matrix
Реализация sklearn
FastICA(n_components=20).fit_transform(X)
Описание гиперпараметров

n_components, fun, max_iter.

Интуиция использования

Полезно для сигналов и разделения источников.

Сетка гиперпараметров для гридсёрч
{"ica__n_components":[10,20,50],"ica__fun":["logcosh","exp"],"ica__max_iter":[200,500]}

NMF

Принцип работы коротко

Неразложение неотрицательной матрицы на базисы и коэффициенты.

Математическое обоснование подробно

Оптимизация X≈WH с ограничением W,H≥0.

Собственная реализация
# multiplicative updates for W,H
Реализация sklearn
NMF(n_components=20, init="nndsvda").fit_transform(X_nonneg)
Описание гиперпараметров

n_components, init, alpha_W.

Интуиция использования

Интерпретируемые темы/факторы из неотрицательных данных.

Сетка гиперпараметров для гридсёрч
{"nmf__n_components":[10,20,40],"nmf__init":["random","nndsvda"],"nmf__alpha_W":[0.0,0.1]}

Linear Discriminant Analysis (как reduction)

Принцип работы коротко

Проекция, максимизирующая разделимость классов.

Математическое обоснование подробно

Оптимизирует отношение межклассовой и внутриклассовой дисперсий.

Собственная реализация
# solve generalized eigenproblem inv(Sw)Sb
Реализация sklearn
LinearDiscriminantAnalysis(n_components=2).fit_transform(X,y)
Описание гиперпараметров

n_components, solver, shrinkage.

Интуиция использования

Когда есть метки и важна компактная, разделяющая проекция.

Сетка гиперпараметров для гридсёрч
{"lda__solver":["svd","lsqr"],"lda__shrinkage":[None,"auto"]}
