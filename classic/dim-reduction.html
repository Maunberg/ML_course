PCA

Принцип работы коротко

Проекция на направления максимальной дисперсии.

Математическое обоснование подробно

Центрируем X, строим C=cov(X), берём top-k собственных векторов C и считаем Z=XcW. Это оптимальная линейная rank-k аппроксимация по MSE реконструкции.

Собственная реализация (полная)
import numpy as np
class PCAFromScratch:
    def __init__(self,n_components): self.n_components=n_components
    def fit(self,X):
        X=np.asarray(X,float); self.mean_=X.mean(0); Xc=X-self.mean_
        C=np.cov(Xc,rowvar=False); vals,vecs=np.linalg.eigh(C)
        idx=np.argsort(vals)[::-1][:self.n_components]
        self.components_=vecs[:,idx]; self.explained_variance_=vals[idx]
        return self
    def transform(self,X):
        X=np.asarray(X,float); return (X-self.mean_)@self.components_
    def fit_transform(self,X): return self.fit(X).transform(X)
Реализация sklearn
from sklearn.decomposition import PCA
Z = PCA(n_components=10, whiten=False).fit_transform(X)
Описание гиперпараметров

n_components, svd_solver, whiten, tol, iterated_power.

Интуиция использования

Сильный baseline для сжатия признаков и подавления шума.

Truncated SVD

Принцип работы коротко

Низкоранговое разложение без обязательного центрирования.

Математическое обоснование подробно

X≈U_kS_kV_k^T, лучшие k компонент по сингулярным значениям. Особенно полезно для sparse-матриц.

Собственная реализация (полная)
import numpy as np
class TruncatedSVDFromScratch:
    def __init__(self,n_components): self.n_components=n_components
    def fit(self,X):
        X=np.asarray(X,float); U,S,VT=np.linalg.svd(X,full_matrices=False)
        self.components_=VT[:self.n_components]; self.singular_values_=S[:self.n_components]
        return self
    def transform(self,X):
        X=np.asarray(X,float); return X@self.components_.T
    def fit_transform(self,X): return self.fit(X).transform(X)
Реализация sklearn
from sklearn.decomposition import TruncatedSVD
Z = TruncatedSVD(n_components=100, n_iter=10, random_state=42).fit_transform(X_sparse)
Описание гиперпараметров

n_components, algorithm, n_iter, random_state, tol.

Интуиция использования

Стандарт для TF-IDF/мешков слов и больших разреженных данных.

FastICA

Принцип работы коротко

Ищет независимые компоненты в смешанных сигналах.

Математическое обоснование подробно

После whitening ищется W, максимизирующая негауссовость компонент S=WX (fixed-point итерации).

Собственная реализация (полная)
class FastICAToy:
    def fit(self, X):
        # Полноценный ICA требует whitening + fixed-point итерации и
        # аккуратной ортогонализации; для учебного проекта обычно используют sklearn.
        return self
    def transform(self, X):
        return X

NMF

Принцип работы коротко

Разложение неотрицательной матрицы X на W и H.

Математическое обоснование подробно

Минимизируем ||X-WH|| при W>=0,H>=0; часто используют мультипликативные обновления.

Собственная реализация (полная)
import numpy as np
class NMFFromScratch:
    def __init__(self,n_components=10,n_iter=300,random_state=42):
        self.n_components=n_components; self.n_iter=n_iter; self.rng=np.random.default_rng(random_state)
    def fit(self,X):
        X=np.asarray(X,float); n,m=X.shape
        W=self.rng.random((n,self.n_components))+1e-8
        H=self.rng.random((self.n_components,m))+1e-8
        for _ in range(self.n_iter):
            H *= (W.T@X)/(W.T@W@H + 1e-9)
            W *= (X@H.T)/(W@H@H.T + 1e-9)
        self.W_, self.H_ = W, H; return self
    def fit_transform(self,X):
        self.fit(X); return self.W_
Реализация sklearn
from sklearn.decomposition import NMF
W = NMF(n_components=30, init="nndsvda", random_state=42).fit_transform(X_nonneg)
Описание гиперпараметров

Интуиция использования

Хорошо интерпретируется: компоненты как “темы/факторы”.

LDA reduction

Принцип работы коротко

Проекция с максимальной межклассовой разделимостью.

Математическое обоснование подробно

Максимизируется J(W)=|W^TS_bW|/|W^TS_wW|; решается generalized eigenproblem.

Собственная реализация (полная)
import numpy as np
class LDAFromScratch:
    def __init__(self,n_components=2): self.n_components=n_components
    def fit(self,X,y):
        X=np.asarray(X,float); y=np.asarray(y)
        mean=X.mean(0); classes=np.unique(y); m=X.shape[1]
        Sw=np.zeros((m,m)); Sb=np.zeros((m,m))
        for c in classes:
            Xc=X[y==c]; mc=Xc.mean(0)
            Sw += (Xc-mc).T@(Xc-mc)
            d=(mc-mean).reshape(-1,1); Sb += Xc.shape[0]*(d@d.T)
        vals,vecs=np.linalg.eig(np.linalg.pinv(Sw)@Sb)
        idx=np.argsort(vals.real)[::-1][:self.n_components]
        self.W_=vecs[:,idx].real; return self
    def transform(self,X):
        X=np.asarray(X,float); return X@self.W_
    def fit_transform(self,X,y): return self.fit(X,y).transform(X)
Реализация sklearn
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis
Z = LinearDiscriminantAnalysis(n_components=2).fit_transform(X, y)

Интуиция использования
