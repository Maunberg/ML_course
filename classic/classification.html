Gaussian Naive Bayes

Принцип работы коротко

Оцениваем вероятность класса по формуле Байеса и выбираем класс с максимальным апостериорным лог-скором.

Математическое обоснование подробно

P(y|x) ∝ P(y)∏_j P(x_j|y). Для непрерывных признаков часто предполагается гауссовская плотность: P(x_j|y=c)=N(μ_{cj},σ^2_{cj}). На практике считаем логарифмы: log P(y=c)+Σ_j log N(x_j;μ_{cj},σ^2_{cj}). Обучение: оценка μ,σ^2 и приоров P(y) по выборке MLE-оценками. Пример: если у класса 1 среднее признака выше, то точка с высоким значением признака даст более высокий лог-скор для класса 1.

Собственная реализация (полная, без sklearn)
import numpy as np

class GaussianNBFromScratch:
    def __init__(self, var_smoothing=1e-9):
        self.var_smoothing = var_smoothing

    def fit(self, X, y):
        X = np.asarray(X, dtype=float)
        y = np.asarray(y)
        self.classes_ = np.unique(y)
        n_samples, n_features = X.shape
        self.class_log_prior_ = np.zeros(len(self.classes_))
        self.theta_ = np.zeros((len(self.classes_), n_features))
        self.var_ = np.zeros((len(self.classes_), n_features))

        for i, c in enumerate(self.classes_):
            Xc = X[y == c]
            self.class_log_prior_[i] = np.log(len(Xc) / n_samples)
            self.theta_[i, :] = Xc.mean(axis=0)
            self.var_[i, :] = Xc.var(axis=0) + self.var_smoothing
        return self

    def _joint_log_likelihood(self, X):
        X = np.asarray(X, dtype=float)
        jll = []
        for i in range(len(self.classes_)):
            mean = self.theta_[i]
            var = self.var_[i]
            log_prob = -0.5 * np.sum(np.log(2.0 * np.pi * var))
            log_prob -= 0.5 * np.sum(((X - mean) ** 2) / var, axis=1)
            jll.append(self.class_log_prior_[i] + log_prob)
        return np.vstack(jll).T

    def predict(self, X):
        jll = self._joint_log_likelihood(X)
        return self.classes_[np.argmax(jll, axis=1)]

    def predict_proba(self, X):
        jll = self._joint_log_likelihood(X)
        jll = jll - np.max(jll, axis=1, keepdims=True)
        probs = np.exp(jll)
        probs /= probs.sum(axis=1, keepdims=True)
        return probs
Реализация sklearn
from sklearn.naive_bayes import GaussianNB
model = GaussianNB(var_smoothing=1e-9)
model.fit(X_train, y_train)
y_pred = model.predict(X_test)
Описание гиперпараметров (максимально подробно)
  • var_smoothing: добавка к дисперсии; стабилизирует вычисления, особенно при почти константных признаках.
  • priors: ручная установка априорных вероятностей классов; полезно при известном реальном дисбалансе.
  • Практика: сначала стандартизуй признаки и подбирай var_smoothing лог-шкалой.
Интуиция использования

Отличный быстрый baseline и сильный выбор для текстовых/разреженных задач. Ограничение: независимость признаков часто нарушается.

K-Nearest Neighbors

Принцип работы коротко

Новый объект получает класс по большинству среди k ближайших объектов train-набора.

Математическое обоснование подробно

Для точки x ищем множество соседей N_k(x) по метрике d. Решение: ŷ=argmax_c Σ_{i∈N_k(x)} w_i I(y_i=c), где w_i - вес (равный или обратный расстоянию). Обучения параметров как в линейных моделях нет: модель “учится” запоминанием train-данных. Важны масштабирование и выбор метрики.

Собственная реализация (полная, без sklearn)
import numpy as np
from collections import Counter

class KNNClassifierFromScratch:
    def __init__(self, n_neighbors=5, weights="uniform", p=2):
        self.n_neighbors = n_neighbors
        self.weights = weights
        self.p = p

    def fit(self, X, y):
        self.X_train = np.asarray(X, dtype=float)
        self.y_train = np.asarray(y)
        return self

    def _distance(self, x):
        return np.sum(np.abs(self.X_train - x) ** self.p, axis=1) ** (1 / self.p)

    def predict(self, X):
        X = np.asarray(X, dtype=float)
        preds = []
        for x in X:
            dists = self._distance(x)
            idx = np.argsort(dists)[:self.n_neighbors]
            labels = self.y_train[idx]
            if self.weights == "distance":
                w = 1.0 / (dists[idx] + 1e-12)
                score = {}
                for lbl, ww in zip(labels, w):
                    score[lbl] = score.get(lbl, 0.0) + ww
                preds.append(max(score, key=score.get))
            else:
                preds.append(Counter(labels).most_common(1)[0][0])
        return np.array(preds)
Реализация sklearn
from sklearn.neighbors import KNeighborsClassifier
model = KNeighborsClassifier(n_neighbors=7, metric="minkowski", p=2, weights="distance")
model.fit(X_train, y_train)
y_pred = model.predict(X_test)
Описание гиперпараметров (максимально подробно)
  • n_neighbors: смещение/дисперсия; маленький k -> шумочувствительность, большой k -> сглаживание.
  • weights: uniform или distance; второй вариант полезен, когда плотность данных неоднородна.
  • metric, p: геометрия пространства; для sparse/высоких размерностей выбор критичен.
  • algorithm: auto/ball_tree/kd_tree/brute; влияет на скорость, не на качество.
  • leaf_size: баланс памяти и скорости в деревьях поиска соседей.

Decision Tree

Математическое обоснование подробно

На каждом узле выбираем сплит, максимизирующий уменьшение неоднородности: ΔI = I(parent)-Σ(n_j/n)I(child_j). Для классификации обычно Gini = 1-Σp_c^2 или Entropy = -Σp_c log p_c. Обучение - жадное рекурсивное построение дерева до критериев остановки.

Собственная реализация (полная, без sklearn)
import numpy as np

class DecisionTreeNode:
    def __init__(self, feature=None, threshold=None, left=None, right=None, value=None):
        self.feature = feature
        self.threshold = threshold
        self.left = left
        self.right = right
        self.value = value

class DecisionTreeClassifierFromScratch:
    def __init__(self, max_depth=10, min_samples_split=2):
        self.max_depth = max_depth
        self.min_samples_split = min_samples_split

    def _gini(self, y):
        _, counts = np.unique(y, return_counts=True)
        p = counts / len(y)
        return 1 - np.sum(p ** 2)

    def _best_split(self, X, y):
        best_gain = -1
        best = (None, None)
        parent_imp = self._gini(y)
        n, m = X.shape
        for f in range(m):
            thresholds = np.unique(X[:, f])
            for t in thresholds:
                left_idx = X[:, f] <= t
                right_idx = ~left_idx
                if left_idx.sum() == 0 or right_idx.sum() == 0:
                    continue
                left_imp = self._gini(y[left_idx])
                right_imp = self._gini(y[right_idx])
                gain = parent_imp - (left_idx.sum()/n)*left_imp - (right_idx.sum()/n)*right_imp
                if gain > best_gain:
                    best_gain = gain
                    best = (f, t)
        return best

    def _build(self, X, y, depth):
        classes, counts = np.unique(y, return_counts=True)
        majority = classes[np.argmax(counts)]
        if depth >= self.max_depth or len(y) < self.min_samples_split or len(classes) == 1:
            return DecisionTreeNode(value=majority)
        f, t = self._best_split(X, y)
        if f is None:
            return DecisionTreeNode(value=majority)
        left_idx = X[:, f] <= t
        right_idx = ~left_idx
        left = self._build(X[left_idx], y[left_idx], depth + 1)
        right = self._build(X[right_idx], y[right_idx], depth + 1)
        return DecisionTreeNode(feature=f, threshold=t, left=left, right=right)

    def fit(self, X, y):
        X = np.asarray(X, dtype=float)
        y = np.asarray(y)
        self.root = self._build(X, y, 0)
        return self

    def _predict_one(self, x, node):
        if node.value is not None:
            return node.value
        if x[node.feature] <= node.threshold:
            return self._predict_one(x, node.left)
        return self._predict_one(x, node.right)

    def predict(self, X):
        X = np.asarray(X, dtype=float)
        return np.array([self._predict_one(x, self.root) for x in X])
Гиперпараметры (подробно)
  • criterion: gini/entropy/log_loss - функция качества сплита.
  • max_depth: главный ограничитель сложности.
  • min_samples_split, min_samples_leaf: минимальный размер узла/листа.
  • max_features: регуляризация через ограничение признаков.
  • ccp_alpha: post-pruning (cost-complexity pruning).

Random Forest

Математическое обоснование подробно

Каждое дерево обучается на бутстрэп-выборке, а в узлах рассматривается случайное подмножество признаков. Это декоррелирует деревья и снижает дисперсию ансамбля при голосовании.

Собственная реализация (полная, без sklearn)
import numpy as np
from collections import Counter

class RandomForestClassifierFromScratch:
    def __init__(self, n_estimators=50, max_depth=8, max_features="sqrt", random_state=42):
        self.n_estimators = n_estimators
        self.max_depth = max_depth
        self.max_features = max_features
        self.rng = np.random.default_rng(random_state)

    def _feature_count(self, n_features):
        if self.max_features == "sqrt":
            return max(1, int(np.sqrt(n_features)))
        if self.max_features == "log2":
            return max(1, int(np.log2(n_features)))
        return n_features

    def fit(self, X, y):
        X = np.asarray(X, dtype=float)
        y = np.asarray(y)
        n_samples, n_features = X.shape
        self.trees_ = []
        self.feat_idx_ = []
        k = self._feature_count(n_features)

        for _ in range(self.n_estimators):
            idx = self.rng.integers(0, n_samples, n_samples)
            fidx = self.rng.choice(n_features, size=k, replace=False)
            tree = DecisionTreeClassifierFromScratch(max_depth=self.max_depth)
            tree.fit(X[idx][:, fidx], y[idx])
            self.trees_.append(tree)
            self.feat_idx_.append(fidx)
        return self

    def predict(self, X):
        X = np.asarray(X, dtype=float)
        all_preds = []
        for tree, fidx in zip(self.trees_, self.feat_idx_):
            all_preds.append(tree.predict(X[:, fidx]))
        all_preds = np.array(all_preds).T
        return np.array([Counter(row).most_common(1)[0][0] for row in all_preds])
Гиперпараметры (подробно)
  • n_estimators: больше деревьев -> стабильнее, но медленнее.
  • max_depth, min_samples_leaf: контролируют переобучение деревьев.
  • max_features: ключ к декорреляции деревьев.
  • bootstrap: включение/выключение бутстрэпа.
  • class_weight: важно при дисбалансе.
  • oob_score: оценка качества out-of-bag без отдельной валидации.

Logistic Regression

Математическое обоснование подробно

p(y=1|x)=σ(w^Tx+b), где σ(z)=1/(1+e^{-z}). Минимизируем log-loss: L=-1/n Σ[y log p + (1-y) log(1-p)] + λR(w). Градиенты: ∂L/∂w = X^T(p-y)/n + λ∂R/∂w, ∂L/∂b = mean(p-y).

Собственная реализация (полная, без sklearn)
import numpy as np

class LogisticRegressionFromScratch:
    def __init__(self, lr=0.05, n_iter=3000, l2=0.0):
        self.lr = lr
        self.n_iter = n_iter
        self.l2 = l2

    def _sigmoid(self, z):
        return 1 / (1 + np.exp(-z))

    def fit(self, X, y):
        X = np.asarray(X, dtype=float)
        y = np.asarray(y, dtype=float)
        n, m = X.shape
        self.w = np.zeros(m)
        self.b = 0.0
        for _ in range(self.n_iter):
            z = X @ self.w + self.b
            p = self._sigmoid(z)
            dw = (X.T @ (p - y)) / n + self.l2 * self.w
            db = np.mean(p - y)
            self.w -= self.lr * dw
            self.b -= self.lr * db
        return self

    def predict_proba(self, X):
        X = np.asarray(X, dtype=float)
        return self._sigmoid(X @ self.w + self.b)

    def predict(self, X):
        return (self.predict_proba(X) >= 0.5).astype(int)
Гиперпараметры (подробно)
  • C: обратная сила регуляризации (меньше C -> сильнее regularization).
  • penalty: l1/l2/elasticnet/None; влияет на sparsity и устойчивость.
  • solver: совместимость с penalty и размерностью (liblinear, lbfgs, saga).
  • class_weight: компенсация дисбаланса.
  • max_iter, tol: критерии сходимости.
  • multi_class: one-vs-rest или multinomial softmax.
