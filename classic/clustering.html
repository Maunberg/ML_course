KMeans

Принцип работы коротко

Назначает точки ближайшему центроиду и обновляет центры.

Математическое обоснование подробно

Минимизация Σ||x-μ_k||^2.

Собственная реализация
# assign -> update -> repeat
Реализация sklearn
KMeans(n_clusters=4).fit(X)
Описание гиперпараметров

n_clusters, init, n_init.

Интуиция использования

Для компактных, примерно сферических кластеров.

Сетка гиперпараметров для гридсёрч
{"n_clusters":[2,3,4,5,6],"init":["k-means++","random"],"n_init":[10,20]}

DBSCAN

Принцип работы коротко

Находит плотные регионы и шум.

Математическое обоснование подробно

Core points: минимум соседей в радиусе eps.

Собственная реализация
# neighborhood expansion
Реализация sklearn
DBSCAN(eps=0.5,min_samples=5).fit(X)
Описание гиперпараметров

eps, min_samples.

Интуиция использования

Кластеры произвольной формы и наличие шума.

Сетка гиперпараметров для гридсёрч
{"eps":[0.2,0.5,0.8],"min_samples":[3,5,10]}

Agglomerative

Принцип работы коротко

Иерархически сливает кластеры от точек вверх.

Математическое обоснование подробно

Выбор пар по linkage: ward/average/complete.

Собственная реализация
# distance matrix and merges
Реализация sklearn
AgglomerativeClustering(n_clusters=4).fit(X)
Описание гиперпараметров

n_clusters, linkage, metric.

Интуиция использования

Полезна для дендрограмм и анализа структуры данных.

Сетка гиперпараметров для гридсёрч
{"n_clusters":[2,3,4,5],"linkage":["ward","complete","average"]}

Gaussian Mixture

Принцип работы коротко

Моделирует данные смесью гауссиан, soft-кластеризация.

Математическое обоснование подробно

EM-алгоритм: E-шаг вероятности компонент, M-шаг обновление параметров.

Собственная реализация
# initialize mu,sigma,pi -> EM iterations
Реализация sklearn
GaussianMixture(n_components=4).fit(X)
Описание гиперпараметров

n_components, covariance_type, reg_covar.

Интуиция использования

Когда важны вероятности принадлежности к кластерам.

Сетка гиперпараметров для гридсёрч
{"n_components":[2,3,4,5],"covariance_type":["full","diag"],"reg_covar":[1e-6,1e-4]}

BIRCH

Принцип работы коротко

Строит CF-дерево и кластеризует сжатое представление.

Математическое обоснование подробно

Пошаговое агрегирование статистик кластера (N, LS, SS).

Собственная реализация
# CF tree insertions with threshold
Реализация sklearn
Birch(n_clusters=4, threshold=0.5).fit(X)
Описание гиперпараметров

threshold, branching_factor, n_clusters.

Интуиция использования

Быстрая кластеризация крупных датасетов.

Сетка гиперпараметров для гридсёрч
{"threshold":[0.3,0.5,0.7],"branching_factor":[25,50,100],"n_clusters":[None,3,5]}
