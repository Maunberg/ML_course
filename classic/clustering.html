KMeans

Принцип работы коротко

Назначает точки ближайшему центроиду и пересчитывает центроиды.

Математическое обоснование подробно

Минимизируется J=sum_i||x_i-mu_{c_i}||^2. Lloyd: assign/update до сходимости.

Собственная реализация (полная)
import numpy as np
class KMeansFromScratch:
    def __init__(self,n_clusters=3,max_iter=300,random_state=42):
        self.n_clusters=n_clusters; self.max_iter=max_iter; self.rng=np.random.default_rng(random_state)
    def fit(self,X):
        X=np.asarray(X,float); n=len(X); self.centroids_=X[self.rng.choice(n,self.n_clusters,replace=False)]
        for _ in range(self.max_iter):
            d=((X[:,None,:]-self.centroids_[None,:,:])**2).sum(2)
            labels=d.argmin(1)
            new=np.array([X[labels==k].mean(0) if np.any(labels==k) else self.centroids_[k] for k in range(self.n_clusters)])
            if np.allclose(new,self.centroids_): break
            self.centroids_=new
        self.labels_=labels; return self
    def predict(self,X):
        X=np.asarray(X,float); d=((X[:,None,:]-self.centroids_[None,:,:])**2).sum(2)
        return d.argmin(1)
Реализация sklearn
from sklearn.cluster import KMeans
labels = KMeans(n_clusters=4, n_init=20, random_state=42).fit_predict(X)
Описание гиперпараметров

n_clusters, init, n_init, max_iter, tol, algorithm.

Интуиция использования

Для компактных кластеров и стандартизованных признаков.

DBSCAN

Принцип работы коротко

Формирует кластеры из плотных областей, шум исключает.

Математическое обоснование подробно

Core point: >=min_samples соседей в eps-радиусе. Кластер = компонента достижимости по плотности.

Agglomerative Clustering

Принцип работы коротко

Иерархически сливает кластеры от одиночных точек к крупным группам.

Математическое обоснование подробно

На каждом шаге объединяется пара кластеров с минимальной связью по выбранному linkage.

Собственная реализация (полная)
# Полная реализация требует поддержки матрицы расстояний и истории merge.
# В учебных задачах обычно берут scipy/sklearn реализацию.
Реализация sklearn
from sklearn.cluster import AgglomerativeClustering
labels = AgglomerativeClustering(n_clusters=4, linkage="ward").fit_predict(X)
Описание гиперпараметров

n_clusters, distance_threshold, linkage, metric.

Интуиция использования

Полезна для анализа иерархии и дендрограмм.

Сетка гиперпараметров для гридсёрч
{"n_clusters":[2,3,4,5],"linkage":["ward","complete","average"],"metric":["euclidean","manhattan"]}

Gaussian Mixture Model

Принцип работы коротко

Моделирует данные смесью гауссиан и даёт soft-присвоение кластера.

Математическое обоснование подробно

p(x)=sum_k pi_k N(x|mu_k,Sigma_k). EM: E-шаг рассчитывает gamma_ik, M-шаг обновляет параметры смеси.

Собственная реализация (полная)
# Полный EM включает инициализацию (pi,mu,Sigma), E/M шаги и контроль log-likelihood.
Реализация sklearn
from sklearn.mixture import GaussianMixture
gmm = GaussianMixture(n_components=4, covariance_type="full", random_state=42).fit(X)
labels = gmm.predict(X)
Описание гиперпараметров

n_components, covariance_type, reg_covar, n_init, max_iter, tol.

Интуиция использования

Когда важны вероятности принадлежности к кластерам.

Сетка гиперпараметров для гридсёрч
{"n_components":[2,3,4,5],"covariance_type":["full","diag"],"reg_covar":[1e-6,1e-4],"n_init":[1,5]}

BIRCH

Принцип работы коротко

Сжимает данные в CF-дерево и кластеризует компактные сводки.

Математическое обоснование подробно

Каждый CF узел хранит (N,LS,SS), что позволяет агрегировать подкластерную структуру при малых затратах памяти.

Собственная реализация (полная)
# Полный CF-tree с insert/split достаточно объёмен; для практики используют sklearn реализацию.
Реализация sklearn
from sklearn.cluster import Birch
labels = Birch(threshold=0.5, branching_factor=50, n_clusters=4).fit_predict(X)
Описание гиперпараметров

threshold, branching_factor, n_clusters, compute_labels.

Интуиция использования

Подходит для больших датасетов, где KMeans слишком дорогой.

Сетка гиперпараметров для гридсёрч
{"threshold":[0.3,0.5,0.7],"branching_factor":[25,50,100],"n_clusters":[None,3,5]}
