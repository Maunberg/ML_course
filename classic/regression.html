Linear Regression

Принцип работы коротко

Линейная зависимость между признаками и целью.

Математическое обоснование подробно

w=(X^TX)^{-1}X^Ty или GD по MSE.

Собственная реализация
w = np.linalg.pinv(X) @ y
Реализация sklearn
LinearRegression().fit(X_train,y_train)
Описание гиперпараметров

fit_intercept, positive.

Интуиция использования

Быстрый и интерпретируемый baseline.

Сетка гиперпараметров для гридсёрч
{"fit_intercept":[True,False],"positive":[False,True]}

Ridge

Принцип работы коротко

L2-регуляризация линейной регрессии.

Математическое обоснование подробно

||y-Xw||^2 + α||w||^2.

Собственная реализация
w = np.linalg.inv(X.T@X + a*np.eye(X.shape[1])) @ X.T @ y
Реализация sklearn
Ridge(alpha=1.0).fit(X_train,y_train)
Описание гиперпараметров

alpha, solver.

Интуиция использования

Стабилизирует модель при коррелирующих признаках.

Сетка гиперпараметров для гридсёрч
{"alpha":[0.01,0.1,1,10,100],"solver":["auto","svd","sag"]}

Lasso

Принцип работы коротко

L1-регуляризация зануляет часть коэффициентов.

Математическое обоснование подробно

||y-Xw||^2 + α||w||_1.

Собственная реализация
# Coordinate descent for L1 penalty
Реализация sklearn
Lasso(alpha=0.01).fit(X_train,y_train)
Описание гиперпараметров

alpha, max_iter, selection.

Интуиция использования

Удобна для отбора признаков.

Сетка гиперпараметров для гридсёрч
{"alpha":[1e-4,1e-3,1e-2,1e-1],"max_iter":[1000,3000,5000]}

ElasticNet

Принцип работы коротко

Комбинация L1+L2 регуляризации.

Математическое обоснование подробно

||y-Xw||^2 + α(ρ||w||_1 + (1-ρ)||w||_2^2).

Собственная реализация
# Proximal gradient / coordinate descent
Реализация sklearn
ElasticNet(alpha=0.01,l1_ratio=0.5).fit(X_train,y_train)
Описание гиперпараметров

alpha, l1_ratio.

Интуиция использования

Баланс между стабильностью и отбором признаков.

Сетка гиперпараметров для гридсёрч
{"alpha":[1e-4,1e-3,1e-2,1e-1],"l1_ratio":[0.1,0.5,0.9]}

RandomForestRegressor

Принцип работы коротко

Усредняет предсказания множества деревьев.

Математическое обоснование подробно

Bagging снижает дисперсию базового дерева.

Собственная реализация
# Bootstrap trees and average outputs
Реализация sklearn
RandomForestRegressor(n_estimators=300).fit(X_train,y_train)
Описание гиперпараметров

n_estimators, max_depth, min_samples_leaf.

Интуиция использования

Сильный выбор для табличных нелинейных задач.

Сетка гиперпараметров для гридсёрч
{"n_estimators":[100,300,500],"max_depth":[None,8,16],"min_samples_leaf":[1,2,4]}
