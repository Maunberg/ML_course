Трансформер сеть

Encoder-based Transformer

Принцип работы коротко

Использует self-attention, чтобы каждый токен видел контекст всей последовательности.

Математическое обоснование подробно

Attention(Q,K,V)=softmax(QK^T/sqrt(d_k))V, multi-head attention объединяет несколько проекций внимания.

Собственная реализация
# NumPy/PyTorch: projection Q,K,V -> attention -> FFN -> residual + norm
Реализация sklearn
# В sklearn нет нативного Transformer; обычно PyTorch/HuggingFace.
Описание гиперпараметров

d_model, num_heads, num_layers, d_ff, dropout, lr.

Интуиция использования

Для NLP и мультимодальных задач, когда важны длинные зависимости.
