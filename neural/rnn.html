Рекуррентная сеть (RNN/LSTM/GRU)

LSTM для последовательностей

Принцип работы коротко

Обрабатывает последовательность шаг за шагом, сохраняя скрытое состояние и память.

Математическое обоснование подробно

LSTM использует ворота forget/input/output для контроля потока информации и градиентов.

Собственная реализация
# RNN-cell/LSTM-cell на NumPy: h_t = f(Wx_t + Uh_{t-1} + b)
Реализация sklearn
# В sklearn нет нативной LSTM; обычно PyTorch/Keras.
Описание гиперпараметров

hidden_size, num_layers, dropout, bidirectional, lr.

Интуиция использования

Для временных рядов и текстов, когда важен порядок токенов/наблюдений.
