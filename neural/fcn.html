Полносвязная сеть (FCN/MLP)

MLP Classifier

Принцип работы коротко

Последовательность линейных слоев и нелинейностей обучается через backpropagation.

Математическое обоснование подробно

h=σ(Wx+b), оптимизация функции потерь градиентным спуском через цепное правило.

Собственная реализация
# NumPy MLP: forward -> loss -> backward -> update
Реализация sklearn
from sklearn.neural_network import MLPClassifier
model = MLPClassifier(hidden_layer_sizes=(128,64), max_iter=200).fit(X_train,y_train)
Описание гиперпараметров

hidden_layer_sizes, activation, alpha, learning_rate_init.

Интуиция использования

Работает на табличных и векторных признаках, когда линейных моделей недостаточно.
